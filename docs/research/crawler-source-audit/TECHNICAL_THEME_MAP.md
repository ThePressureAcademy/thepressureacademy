# TECHNICAL_THEME_MAP

High-level map of recurring **public** technical themes observed across credible accepted sources. This is **not** a curriculum — it is a co-occurrence matrix Codex can use to choose which systems are well-supported by public material and which are gap-driven.

A theme is listed when it appears, by name or by function, in ≥ 3 credible public sources from `SOURCE_BIBLIOGRAPHY.md`. Paid-course sources do **not** count toward the ≥ 3 threshold. Bilibili sources do not count.

`Public Source Examples` cites the strongest representative free sources. `Notes for Codex` is a flag for the synthesis phase, not technical detail.
