## 6. Suggested IF→THEN chain categories for the Blueprint

These categories are derived from the theme map. Codex chooses the exact phrasing and granularity.

- **Standing chain**
  - IF opponent shells (head down, posture broken) → THEN snapdown → front headlock branch
  - IF opponent stands tall → THEN collar tie / wrist control → snap or angle change
  - IF opponent reaches → THEN arm drag → back / single-leg branch
- **Front headlock chain**
  - IF chin exposed → guillotine
  - IF far-side arm in → D'Arce branch
  - IF far-side arm out → anaconda branch
  - IF opponent base sound → go-behind → back take / mat return
  - IF opponent posts → turtle breakdown → back / submission
- **Body lock / mat return chain**
  - IF over-under clinch → body lock takedown branch
  - IF rear body lock secured → knee pinch / knee staple / cross-scissor → mat return → pin or back take
- **Single leg chain**
  - IF head-inside → running-the-pipe / crackdown
  - IF head-outside → high crotch / double-leg finish
  - IF opponent counters with guillotine → exposure-aware exit
- **Scramble-to-control chain**
  - IF scramble triggered → front-headlock recapture / back take / pin recovery
- **Leg-lock chain (basic)**
  - IF feet-on-hips → single-leg-X → straight ankle lock / ashi family
  - IF same-side leg control → outside ashi
- **Leg-lock chain (intermediate)**
  - IF cross-leg control → saddle / inside sankaku entry (mark gap; original synthesis)
  - IF 50/50 reached → entry-aware finish / escape (mark gap; original synthesis)
- **Safety chain (mandatory)**
  - IF heel-hook position reached → controlled application; tap-on-grip in training; never crank
  - IF caught in leg lock → heel-hide / hip-escape / free-leg defense
